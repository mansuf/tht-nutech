# Take Home Test - PT Nutech Integrasi

## Deskripsi

Project Take Home Test untuk PT Nutech Integrasi untuk Role Backend Developer

## Run the app

Clone the repository

```bash
git clone https://github.com/mansuf/tht-nutech.git
cd tht-nutech
```

Install dependencies (i use yarn)

```bash
yarn

# or

npm install
```

Setup environment variables

```bash
cp .env.example .env

# Proceed to setup environment variables such as PORT, JWT_SECRET, and POSTGRES_URL
```

Run the website application

```bash
yarn start

# or 

npm start
```
